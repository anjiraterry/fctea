# fctea | Abuja City Operating System

A modern city discovery platform for Abuja that combines local places, events, creators, news, culture, and trending hotspots.

## Tech Stack
- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS, shadcn/ui, Framer Motion
- **Backend**: Prisma ORM, PostgreSQL (via Supabase)
- **Auth**: NextAuth.js
- **State Management**: Zustand, TanStack Query

## Core Features
- **Places**: Discovery and curated listings.
- **Events**: Upcoming city events and ticketing.
- **News & Blog**: Editorial content and city updates.
- **People**: Profiles for creators, DJs, and local legends.
- **Brands**: Dedicated presence for local and global brands.
- **Admin Dashboard**: Content management and "Hot Score" system.
- **Inline Editing**: Live content updates for editors.

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
